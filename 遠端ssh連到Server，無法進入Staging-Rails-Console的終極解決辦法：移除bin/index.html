情境：

當用ssh遠端連線至server時，要進入專案內的rails console.log，出現rails new [options]，找不到rails project的目錄，紀錄如何解決的筆記。

解決辦法

  1. 在local端用力打開 config/deploy.rb
  2. 找到linked_dirs，然後把bin吃掉
